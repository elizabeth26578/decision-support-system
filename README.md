# Decision Support System

Цей проєкт є простою системою підтримки прийняття рішень, яка була розроблена в рамках навчального завдання.

Метою системи є вибір найбільш оптимального ноутбука для студента-програміста на основі кількох критеріїв.

У системі використовуються:

- альтернативи (моделі ноутбуків)
- критерії оцінювання (ціна, продуктивність, автономність, вага, гарантія)
- матриця оцінювання
- обчислення інтегральної оцінки альтернатив

Для аналізу альтернатив реалізовано два методи:

- Weighted Sum Model
- TOPSIS

Програма обчислює підсумкову оцінку кожної альтернативи, формує рейтинг та визначає найбільш оптимальний варіант. Також передбачено можливість відобразити результати у вигляді графіка.

## Структура проєкту

app/
- controllers — взаємодія з користувачем
- services — реалізація алгоритмів прийняття рішень
- data — робота з даними
- models — моделі альтернатив та критеріїв

data/
- sample_data.json — приклад вхідних даних

main.py — точка запуску програми

## Запуск програми

1. Встановити необхідні бібліотеки

pip3 install -r requirements.txt

2. Запустити програму

python3 main.py

Після запуску програми відкриється консольне меню, де можна переглядати дані, будувати рейтинг альтернатив та відображати результати у вигляді графіка.

## Висновок
Було розроблено програмний продукт класу СППР для вибору ноутбука.  
Система працює з альтернативами, критеріями, матрицею оцінювання, реалізує аналітичний блок, формує рейтинг і пояснює вибір.  
Проєкт придатний для реальної задачі та може бути легко розширений для інших предметних областей.
